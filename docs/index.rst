**Random**
---------

**Random Nekos:**
``nbapi.random.neko()``
  -Options:
    `source=True/False` - Shows image source and ID (for reporting)

**Random Anime:**
``nbapi.random.anime()``
  -Options:
    `source=True/False` - Shows image source and ID (for reporting)







**Search**
--------
**Nekos:**
``nbapi.search.neko(*int)``
  -Options:
    `source=True/False` - Shows image source and ID (for reporting)

**Anime:**
``nbapi.search.anime(*int)``
  -Options:
    `source=True/False` - Shows image source and ID (for reporting)
